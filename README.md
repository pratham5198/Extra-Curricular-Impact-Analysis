# Impact of Extracurricular Activities on Personality Transformation

This project explores the relationship between students' involvement in extracurricular activities and their academic performance and personality development.

## Project Structure
- `Statistical_Project.ipynb`: Jupyter Notebook containing data analysis and visualization.
- `Do extracurricular activities help transform personality (1).csv`: The dataset used for analysis.

## Key Insights
- Analyzes the correlation between age and academic performance.
- Investigates the most popular extracurricular activities (Sports, Art, Technology).
- Evaluates student opinions on whether these activities should be compulsory.

## How to Run
1. Clone this repository.
2. Ensure you have `pandas`, `numpy`, and `matplotlib` installed.
3. Run the Jupyter Notebook to view the analysis.
